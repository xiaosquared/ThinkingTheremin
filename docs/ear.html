Thinking Through the Theremin

Chapter 2 Ear Training

(super in-progress)

Stage 1 - have a solid auditory image of every interval. Being able to instantly recognize them in a piece of music, also knowing how to produce them. Good way to test - sing! Goal is to be able to start on any note and be able to instantly sing any interval up or down.

How do we get there? One well-known technique is to make a list of songs that start with each interval. Below is my list. Going up is not the same audio image as going down, so it’s important to find a song for the ascending and a song for the descending version of each interval. There are free tools to help you find songs in your preferred genre. Listen to the songs on repeat, sing along. Doesn’t matter if you’re not a good singer. Try startingg in different keys.

Some intervals are easier than others. Spend more time on the ones that are difficult for you. Good to make a chart keeping track of where you are for each interval. Try to randomly quiz yourself during the day by trying to sing an interval. Do it when walking around, when during chores.

Stage 2: Intervals occur in different contexts, and the context changes the way that they sound. For example… major 3rd and then minor 3rd, vs minor 3rd and then major 3rd. Different harmonies! Each song in the list has a harmonic context in which the featured interval appear.

Once you have each individual interval solidly in your mind, try combining. Take an interval, and try to sing it after singing some other interval. To sing the second interval, you kind of need to reorient yourself, which takes too long. Much better to encode the sequence as a unit. Recognize the harmony. Can find songs for each. Maybe it’s tricky to find songs that start with a super specific pattern. I think of segments in songs. Make some charts of examples!!
